# Exercícios de Vetores em JavaScript

---

# Sobre

Este projeto contém exercícios básicos de vetores em JavaScript, utilizando:

- Vetores
- Estruturas de repetição
- Condicionais
- Soma de valores
- Remoção de valores duplicados

---

# Exercícios

## 1️. Soma dos números pares

O programa lê 10 números inteiros, armazena em um vetor e calcula a soma apenas dos números pares.

### Conceitos utilizados

- Vetores
- Laço `for`
- Condicional `if`

---

## 2️. Vetor preenchido automaticamente

O programa preenche automaticamente um vetor com 7 números e exibe todos os valores armazenados.

### Conceitos utilizados

- Vetores
- Atribuição de valores
- Estrutura de repetição

---

## 3️. Quantidade de vezes que um número aparece

O programa lê 10 números inteiros, solicita um número ao usuário e verifica quantas vezes ele aparece no vetor.

### Conceitos utilizados

- Vetores
- Contador
- Comparação de valores
- Condicional `if`

---

## 4️. Vetor sem valores duplicados

O programa lê 10 números e cria um novo vetor sem elementos repetidos.

### Conceitos utilizados

- Comparação entre vetores
- Estrutura de repetição
- Verificação de valores duplicados

---

# Tecnologias utilizadas

- JavaScript
- Console do navegador
